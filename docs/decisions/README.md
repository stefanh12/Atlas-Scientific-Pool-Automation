# Decision Logs

Use this folder for short architecture and heuristic decisions that affect safety or automation behavior.

## When to add an entry

Add a decision entry when changing:

- ORP automation trigger or hysteresis behavior
- level automation trigger/timeout behavior
- dose safety validation order or limits
- pH-effect learning logic
- coordinator payload contract consumed by platforms

## File naming

Use increasing numeric prefixes:

- `0001-short-title.md`
- `0002-short-title.md`

## Required entry format

Use the template in `0000-template.md`.
Keep entries concise and implementation-focused.
